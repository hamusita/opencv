# README
　このディレクトリは画像解析の実験に関するファイルが入っています

## 使い方 
　使い方はそれぞれ以下の通りです
　カレントディレクトリはこのファイルがある階層としてください
 '''
    $ python interface.py
 '''
 '''
    $ python face.py
 '''
 '''
    $ python movie.py
 '''

## 依存関係
　今回作成したプログラムは表示んライブラリ以外に
　cv2 , PIL
　に依存しています
　もし実行時にエラーが出たならば
 '''
    $ pip install opencv-python
 '''
 '''
    $ pip install PIL
 '''
　を実行してから再度試してみてください
